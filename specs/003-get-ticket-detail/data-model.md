# Data Model: 티켓 단건 조회 (GET /api/tickets/:id)

`Ticket` 엔터티 전체 필드 정의는 `specs/001-create-ticket/data-model.md`를 따른다. 이 기능은
새 필드나 상태 전이를 추가하지 않고, `id`로 단건을 조회해 그대로 반환한다.

## 조회 규칙

| 조건 | 결과 |
|------|------|
| `id`에 해당하는 티켓이 존재 | 해당 티켓 전체 정보(200) |
| `id`에 해당하는 티켓이 없음(삭제됨 포함) | NOT_FOUND(404) |
| `id`가 정수가 아닌 값 | NOT_FOUND(404) — 존재하지 않는 것과 동일하게 취급 |

이 기능은 읽기 전용이며 어떤 필드도 변경하지 않는다.
